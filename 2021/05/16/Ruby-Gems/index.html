    rack

    gem install rack

    Rack provides a minimal, modular and adaptable interface for developing web applications in Ruby. By wrapping HTTP requests and responses in the simplest way possible, it unifies and distills the API for web servers, web frameworks, and software in between (the so-called middleware) into a single method call.

    Rack提供了一個在用Ruby來開發網路應用程式時的介面。藉由將HTTP requests 和 responses 盡可能地簡單包裹，結合伺服器、框架和軟體為一個簡單的call方法。簡單來說，就是讓Ruby能夠和伺服器之間進行溝通的工具。

    faker

    gem install faker

    Faker, a port of Data::Faker from Perl, is used to easily generate fake data: names, addresses, phone numbers, etc.

    這是測試時常用的套件，可以產生很多假資料，包括姓名、地址、email和電話……等等。

    rspec

    gem install rspec

    這是Test-Driven Development（TDD），測試驅動開發常用的套件，可以在開發時先寫規格，然後用這個套件來跑測試看看code有沒有寫對。

    puma

    gem install puma

    Puma is a simple, fast, threaded, and highly concurrent HTTP 1.1 server for Ruby/Rack applications. Puma is intended for use in both development and production environments.

    雖然它跟某個著名的鞋子品牌同名，但在這裡它是啟動server時的一個好幫手，可以跑的比內建的還快。

    kaminari

    gem install kaminari

    Kaminari is a Scope & Engine based, clean, powerful, agnostic, customizable and sophisticated paginator for Rails 4+

    一個好用的分頁產生器，還可以客製化成自己喜歡的風格樣式。

    nokogiri

    gem install nokogiri

    Nokogiri (鋸) makes it easy and painless to work with XML and HTML from Ruby. It provides a sensible, easy-to-understand API for reading, writing, modifying, and querying documents. It is fast and standards-compliant by relying on native parsers like libxml2 (C) and xerces (Java).

    寫網路爬蟲時很好用的一個套件。

    simple_form

    gem install simple_form

    Forms made easy!

    如同作者的描述，這個套件可以讓我們在寫表單時少寫一些程式碼，讓表單變得更簡單。

    foreman

    gem install foreman

    Process manager for applications with multiple components

    這個套件可以用一個終端機就開啟多個應用。例如要同時開 rails server 和 webpack-dev-server 時，只要將欲開啟的東西寫入Procfile，然後只用一個終端機輸入指令foreman s就可以一次把兩個打開，而不用多開一個終端機。

    devise

    gem install device

    Flexible authentication solution for Rails with Warden

    安裝後就可以立刻做出使用者登出登入的功能，而且功能彈性可以客製化調整，還可以串接第三方登入。

    carrierwave

    gem install carrierwave

    Upload files in your Ruby applications, map them to a range of ORMs, store them on different backends.

    要做上傳檔案的功能時可以使用這個套件，包括圖片、音檔…等，搭配fog-aws套件可以將檔案儲存在Amazon Web Services裡面，實務上好像還蠻常用到的。

    figaro

    gem install figaro

    Simple, Heroku-friendly Rails app configuration using ENV and a single YAML file

    可以將私密的金鑰或密碼等資訊存在環境變數中，並且把該檔案放入.gitignore，讓它不會上傳到GitHub去。

    acts_as_paranoid

    gem install acts_as_paranoid

    A Rails plugin to add soft delete.
    This gem can be used to hide records instead of deleting them, making them recoverable later.

    提供軟刪除功能，讓使用者在刪除資料時只是將該筆資料新增deleted_at欄位的時間，並且隱藏不顯示。
