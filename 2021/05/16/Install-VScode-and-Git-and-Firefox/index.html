    在Windows系統安裝VScode、Git、Firefox

    教學 vscode git firefox
    Created At : 2021-05-16 09:41
    1. 安裝VScode
    2. 安裝Git
    3. 安裝Firefox

    由於電腦硬碟壞掉，所以換了一顆硬碟並重灌，發現有很多資料和軟體要轉移真的麻煩的，而且很多東西必須要重新安裝 Orz

    安裝VScode

    首先到VScode的官方網站，看你的電腦是32位元還是64位元，然後點下去對應的按鈕之後就會開始下載。

    我之前是用portable免安裝版本（.zip那個），所以我只要將VScode的資料夾從壞掉的硬碟複製到好的硬碟即可。

    不過，我發現原本安裝好的擴充套件和snippet沒有跟著轉過來，所以去查了一下。

    原來，VScode的擴充套件位置預設儲存在\Users\你的使用者名稱\.vscode這個資料夾裡，所以把這個資料夾複製貼到重灌後對應的路徑下，再開啟VScode就會出現之前安裝過的套件了！

    另外，snippets檔案則是存在\Users\你的使用者名稱\AppData\Roaming\Code\你的使用者名稱\snippets這個路徑裡面，因為我發現這個Code資料夾還有其他快取和Cookies檔案，所以我是直接把這個Code資料夾也整個複製貼到新硬碟的對應路徑，以確保資料可以完整地轉移。

    安裝Git

    一樣是到Git的官方網站，選擇對應的選項點下去開始下載，然後一路下一步按到底就可以裝好了。

    這邊我一樣是下載免安裝版本（Portable），好處就是刪除時只要把整個資料夾砍掉即可，而要安裝到其他電腦上時只要複製貼上這個資料夾就完成了！

    ……如果這麼簡單的話，我就不會特別寫這篇文章了。

    跟VScode一樣，它還有些設定檔是存在於其他地方的，像這樣直接把資料夾複製過去之後，雖然一樣可以開啟，但之前設定過的username、email、alias……等等方便使用的東西都要重新設定，感覺就很麻煩。

    因此，除了將這個資料夾複製過去（或是其實可以直接在官方下載），還要將\Users\你的使用者名稱\.gitconfig這個檔案也一併複製貼到新硬碟的對應路徑，這樣就不用再重新設定config，也不用擔心會忘記之前的設定而少設了。

    安裝Firefox

    在Firefox官方網站點選「下載Firefox」即可，下載好之後就點兩下開啟安裝檔，然後就是一路下一步。

    這邊想介紹的是，該怎麼把原電腦上的書籤、歷史紀錄、快取……等等資料做轉移。

    在原本磁碟的\User\你的使用者名稱\AppData\Local\Mozilla\Firefox\Profiles這個資料夾裡面，會有一個亂數取名的資料夾（在此稱為A資料夾），而新安裝好firefox的磁碟對應位置應該也會有一個不同亂數的資料夾（在此稱為B資料夾）。

    如果有開啟Firefox瀏覽器的話，先將Firefox完全關閉。

    接著點進去A資料夾，Ctrl+A全選檔案，然後Ctrl+C通通複製，再點進去B資料夾裡面，一樣Ctrl+A全選檔案，接著Ctrl+V全部貼上。

    重新開啟Firefox瀏覽器，此時就可以看到熟悉的分頁、書籤、歷史資料通通都出現了～
