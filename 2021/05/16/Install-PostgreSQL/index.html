    在Windows系統安裝PostgreSQL

    教學 postgresql
    Created At : 2021-05-16 12:22
    1. 安裝
    2. 創建使用者
    3. 創建資料庫
    4. 刪除資料庫
    5. 常用指令
    6. 參考文章

    安裝

    首先，打開安裝好的WSL Ubuntu，然後依序輸入以下內容

    1. $sudo sh -c 'echo "deb http://apt.postgresql.org/pub/repos/apt $(lsb_release -cs)-pgdg main" > /etc/apt/sources.list.d/pgdg.list，輸入完後Enter，接著輸入密碼

    2. $wget --quiet -O - https://www.postgresql.org/media/keys/ACCC4CF8.asc | sudo apt-key add -

    3. $sudo apt-get update

    4. $sudo apt-get -y install postgresql

    5. $sudo apt-get install postgresql-client libpq5 libpq-dev

    到這邊就安裝完成，可以輸入psql --version查詢所裝的PostgreSQL版本。

    創建使用者

    接著要創一個使用者以及資料庫。

    1. $sudo service postgresql start，先啟動PG

    2. $sudo su - postgres

    3. $psql

    4. =#CREATE ROLE your_username WITH LOGIN CREATEDB ENCRYPTED PASSWORD 'your_password'，注意最後密碼的部分要用單引號'不能用雙引號"

    5. =#\q

    6. $su - your_username，並且輸入密碼

    到這裡就建好使用者了

    創建資料庫

    若是要創一個資料庫的話，可以繼續以下步驟

    1. $createdb your_database_name

    2. $sudo su - postgres

    3. $psql

    4. =#\list，可以查看資料庫列表，包含資料庫名稱、擁有者的資訊

    刪除資料庫

    若是要刪除資料庫的話，可以參考以下步驟

    1. =#\q

    2. $su - your_username，並且輸入密碼

    3. $dropdb your_database_name

    4. $sudo su - postgres

    5. $psql

    6. =#\list

    應該會看到剛剛建立的資料庫已經被刪除了

    之後每次開Ubuntu的時候，記得要先輸入步驟6.sudo service postgresql start啟動PG才能使用

    常用指令

    在username@電腦名稱:~$的輸入框下，可以輸入的指令

    指令 說明
    sudo service postgresql start 啟動PostgreSQL
    psql –version 查看PostgreSQL版本
    sudo su - postgres 切換到預設使用者postgres
    • createdb your_database_name
    • dropdb your_database_name

    –

    在postgres@電腦名稱:~$的輸入框下，可以輸入的指令

    指令 說明
    psql –version 查看PostgreSQL版本
    su - username 切換使用者
    psql 進入postgres=#輸入框

    –

    在postgres=#的輸入框下，常用的指令

    指令 說明
    \h 查看所有SQL指令，再輸入q可離開
    \l 列出所有資料庫列表，即\list
    \du 列出所有使用者列表
    \e 打開編輯器
    \q 離開postgres=#輸入框

    這些指令輸入完記得都要有分號;，不然不會動作

    • CREATE ROLE your_username WITH LOGIN CREATEDB ENCRYPTED PASSWORD 'your_password';創建使用者及密碼
    • CREATE DATABASE database_name; Owner為預設的postgres
    • CREATE DATABASE your_database_name OWNER your_username; Owner為your_username
    • DROP DATABASE your_database_name 刪除資料庫
